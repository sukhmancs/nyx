## Partition and mount the drives using disko

```bash
DISK='/dev/disk/by-id/ata-Samsung_SSD_870_EVO_250GB_S6PENL0T902873K'

curl
https://raw.githubusercontent.com/notthebee/nix-config/main/disko/zfs-root/default.nix\
-o /tmp/disko.nix sed -i "s|to-be-filled-during-installation|$DISK|"
/tmp/disko.nix nix --experimental-features "nix-command flakes" run
github:nix-community/disko\
-- --mode disko /tmp/disko.nix
```
